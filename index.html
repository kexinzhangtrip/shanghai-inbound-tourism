一、项目概述

1.1 一句话定位

为德语区（DACH）和韩国游客提供上海本地化深度体验服务，通过社媒内容获客、私域转化、线下交付的闭环模式实现盈利。

1.2 创始人优势

  • 常住上海，熟悉本地生活
  • 精通英语、德语
  • 可拓展韩语线（招募韩国留学生团队）

1.3 商业模式

内容获客 → 私域沉淀 → 服务变现 → 口碑裂变

三、竞争分析

3.1 德语市场

平台现有产品数类型
GetYourGuide2-3个传统景点全日游
Airbnb体验几乎没有—
YouTube德语频道极少非专业内容

结论：蓝海市场，几乎无竞争对手。无美食Tour、无咖啡体验、无夜游、无周边游产品。

3.2 韩国市场

  • 小红书有攻略帖但无服务闭环
  • 平台标注"Korean support"但真正韩语导游极少
  • 本地地接以团队游为主，缺少MZ世代自由行服务

结论：有一定供给但远未满足年轻自由行客群需求。

七、运营模式

7.1 双线协同逻辑

德语线 → 高客单 × 低频 → 利润引擎（创始人亲自交付）

韩语线 → 低客单 × 高频 → 规模引擎（留学生团队交付）

7.2 团队规划

阶段人员角色
第1-3个月创始人1人内容+获客+交付全包
第4-6个月+1-2韩国留学生兼职韩语线带团交付
第7-12个月+1内容助理视频剪辑/发布
第12个月+3-5人小团队获客/交付/运营分工

7.3 人力池：韩国留学生

上海高校（复旦、上外、华师大等）有大量韩国留学生：

  • 中韩双语流利，熟悉上海
  • 时间灵活，时薪 ¥80-150
  • 你做获客和管理，他们做交付 → 韩语线可快速扩容
